# 今日穿搭老虎机

这是一个有趣的老虎机风格的网页应用，用于随机搭配服装、地点和编程语言。

## 功能特性

- 三个独立的转轮：
  - 转轮A：服装类型（JK、女仆装、水手服、COS服、OL装、连衣裙）
  - 转轮B：地点（宿舍、图书馆、咖啡馆、办公室、家里）
  - 转轮C：编程语言/技术（Rust、Java、Python等）
- 按钮点击后，转轮按A→C→B的顺序依次停止
- 流畅的动画效果和现代化的UI设计
- 响应式布局，适配不同屏幕尺寸

## 使用方法

1. 点击"试试手气！"按钮
2. 观察三个转轮旋转
3. 转轮按顺序停止后，查看生成的搭配结果

## 技术栈

- HTML5
- CSS3（包含动画和渐变效果）
- JavaScript（ES6+）

## 文件结构

- `slot-machine.html` - 页面结构
- `slot-machine.css` - 样式和动画
- `slot-machine.js` - 交互逻辑
- `example.html` - 参考示例文件
